Запрос 1
Подсчитаем все записи в таблице сотрудников (employees).

db.employees.aggregate([{$group:{_id:null,count: {$sum:1}}}])

Запрос 2
Нахождение сотрудников, которые интересуются футболом.

db.employees.find({interests:"football"})

Запрос 3
Вывод всех сотрудников со стажем работы по возрастанию

db.employees.aggregate(
   [
     { $sort : { work_exper : 1 } }
   ]
)

Запрос 4
Список всех сотрудников, возраст которых больше 30 и меньше 40.

db.users.find( { age: { $gte: 18, $lte: 30 } } );

Запрос 5
Выведем список имён и возраст сотрудников, которые использовались в нашей базе (не повторяющие значения).

db.employees.distinct("surname")
db.employees.distinct("age")

Запрос 6
Отображение сведений о сотрудников, возраст которых превышает 50 лет.

db.employees.aggregate([{$match:{age:{$gt:50}}}])

Запрос 7
Вывести список сотрудников, которые проживают в Энгельсе.

db.employees.find({ 'address.city': 'Энгельс'})

Запрос 8
Вывести список сотрудников, у которых воинское звание Ефрейтор.

db.employees.aggregate([{$match:{military_rank:'Ефрейтор'}}])
